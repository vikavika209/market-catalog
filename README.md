# Market Catalog

Консольное приложение, имитирующее внутренний каталог товаров маркетплейса.  
Проект реализован на Java, с использованием коллекций и файлового ввода/вывода.

---

## Сборка и запуск

```bash
mvn -q -DskipTests package
java -jar target/market-catalog-core-1.0.0.jar
```

## Основные возможности

CRUD-операции для товаров (создание, просмотр, обновление, удаление)

Поиск и фильтрация по названию, бренду, категории, цене и статусу активности

Кэширование для ускорения повторных запросов

Авторизация пользователей и простое разграничение ролей (Admin/User)

Аудит действий: вход, выход, добавление, изменение и удаление товаров

Хранение данных в локальных CSV-файлах (products.csv, users.csv)

Пагинация в консоли для удобного просмотра длинных списков

Метрики производительности — количество товаров, время последнего запроса, эффективность кэша

JUnit 5 тесты для проверки ключевых сервисов

Автоматическое создание файлов при первом запуске:

products.csv — база товаров

users.csv — список пользователей

audit.log — журнал действий пользователей

---

## Архитектура проекта

Domain layer (доменный слой) — основные сущности (Product, User, Role, Category и др.)

Repository layer (хранилище) — внутренняя коллекция с сохранением данных в CSV

Service layer (сервисный слой) — бизнес-логика (каталог, аутентификация, аудит, кэш, метрики)

UI layer (пользовательский интерфейс) — консольное меню с вводом данных, пагинацией и проверками
